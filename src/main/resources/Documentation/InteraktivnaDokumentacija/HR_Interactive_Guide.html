1. Overview

The HR Dashboard is an application designed to manage employees, projects, and users. It provides users with different roles to handle data and organize workflows efficiently.

Note: The application is created for demonstration purposes and is not intended for production use.

2. Funkcionalnosti

  • Project Management: Add, edit, and delete projects. Define project details like name, description, start/end date, and assigned employees.
  • Employee Management: Add new employees and update their details, including name, position, salary, and assigned projects.
  • User Management: (Superadmin only) Add, edit, and delete users and assign their roles.
  • Security: Users can reset their passwords.
  • User Guide: A dedicated section explains the application's features and how to use them.

3. Korisničke uloge

  • Superadmin: Full access to all functionalities, including managing projects, employees, and users.
  • Admin: Can manage projects and employees but cannot manage users.
  • Default User: Can view project and employee data but cannot modify or manage any data.

5. Tehnički detalji
